# pweb-express-mongodb-P13-2024

## Cara menjalankan

### Buat sebuah file .env berisi Mongo URI, Port, dan JWT Token
```.env
MONGO_URI=<Mongo URI>
PORT=5050
JWT_SECRET=<JWT Token>
```

### Menjalankan NPM
```
npm run build
```
```
npm start
```
